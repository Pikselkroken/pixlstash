REST API

Integrate with scripts, pipelines and external tools. Fetch images, metadata, tags and more.

What can you do with this?

The API makes it easy to plug PixlStash into your own tooling. As an example, our ai-toolkit fork can pull a curated dataset straight from a PixlStash vault and feed it into a LoRA training run — no manual file juggling required.
