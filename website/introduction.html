Getting started with PixlStash

Six short clips covering everything from your first import to exporting a training dataset.

Install now View on GitHub
01 — Import

Drag, drop, done

Import single images, whole folders, ZIP or TAR archives in one go. PixlStash deduplicates and starts tagging your library the moment files land.

02 — Characters

Put names to faces

PixlStash detects every face and groups them by similarity. Assign a character name once and it propagates across your entire library, making face-based sorting instant.

03 — Tagging

Bulk-tag in seconds

The PixlStash tagger run automatically. Review AI predictions, accept or reject with a click, and apply new tags across any selection without leaving the grid.

04 — Smart Score

Surface your best shots

A composite quality score weighs aesthetics and anomaly tags to help you clear the worst images. Filter and sort by smart score to find the real duds.

05 — Search

Find anything by description

CLIP-powered semantic search understands natural language. Type "outdoors golden hour" or "close-up smiling" and get relevant results immediately — no exact tag match needed.

06 — Export

Dataset-ready in one click

Export the current view as a ZIP of images paired with tag or description captions. Choose full images or face crops, pick caption mode, and hand off a clean dataset to your trainer.
