Image grid

Pretty image view with scoring, stacking and sorting.

Project Organisation

Organise your projects efficiently with intuitive controls and additional metadata (here in dark theme)

Similarity sorting

Easily find pictures containing particular characters.

Batch operations

Select images, tag or assign to sets or characters and perform batch actions.

ComfyUI Integration

Run your own ComfyUI workflows directly from PixlStash with progress tracking.

Picture inspection

Review and edit automatic descriptions and tags.

Face detection

Detect and assign faces to people in your photos.

Tag autocompletion

Tag with a keyboard shortcut and let PixlStash suggest tags as you type.

People management

Manage and organize people in your dataset efficiently.

Lots of keyboard shortcuts

Lots of convenient keyboard shortcuts for navigation, selection, scoring, tagging, and more.

Reference folders

Keep your existing folder structure while still benefiting from PixlStash's organisation and metadata features. Manage reference and import folders directly from the UI.

Statistics sidebar

Get a rapid overview of your picture sets, tag distribution, tag prediction confidence, and picture statistics at a glance.

Boolean set operations

Ctrl-click multiple characters or picture sets, then combine them with Union, Overlap, Difference, or Unique (XOR) operations.

Add your own image filter plugins

Extend PixlStash's functionality with custom Python plugins for image processing.

Picture sharing

Share pictures, sets, characters, and projects with a read-only link with no account required for viewers. Tokens are scoped and revocable. Optionally add your own watermarks for branding.

Context menus

Right-click anywhere in the grid or sidebar to rate, tag, assign, move, share, or delete without leaving your current view.

Custom icons for sets

Give each picture set its own coloured icon for instant recognition, no matter how similar the contents look.

Other Features

Automatic tagging, semantic search, and metadata so you can navigate large libraries with fewer steps.

AI tagging

Batch-tag images and faces with state-of-the-art models.

Anomaly detection

Automatically applies a ViT model to detect anomalies in your images, like malformed anatomy.

Automatic grouping

Automatically groups similar images for easier organisation and review.

Plugin system

Extend PixlStash's functionality with custom Python plugins for image processing.

Keyboard shortcuts

Navigate and control PixlStash efficiently using standard keyboard shortcuts for selection, scoring, deletion, etc.

Security notifications

PixlStash checks for known security issues and alerts you in the UI when a fix is available, along with the severity and affected versions.

Faster background processing

Tagging and face extraction use bulk database operations and smarter scheduling for significantly higher throughput. Watch real-time progress in the statistics sidebar.
