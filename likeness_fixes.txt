Likeness improvements (ideas, no implementation):

1) Multi-embedding similarity
- Combine CLIP image embeddings + face embeddings + tag similarity.
- Weighted cosine score: S = w_c * cos(clip) + w_f * cos(face) + w_t * tag_sim.
- Stabilizes across pose/background changes.

2) Face-aware grouping
- If faces detected, prefer face-embedding similarity; fallback to CLIP otherwise.
- Enforce minimum face-count match to avoid unrelated merges.

3) Tag-aware similarity
- Use tag Jaccard or BM25-style similarity to reinforce semantics.
- Penalize low-overlap / contradictory tag sets.

4) Perceptual hash guard
- Add pHash/dHash as a cheap near-duplicate filter or strong boost.

5) Two-stage stacking
- Stage 1: fast approximate neighbors (e.g., FAISS on CLIP).
- Stage 2: re-rank with face + tag similarity.
